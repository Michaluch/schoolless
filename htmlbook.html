GitHub Pages Ергономіка

<html>

Призначення: кореневий елемент HTML-документа

Базові атрибути: lang

Де розміщується: весь документ

<html lang="uk">...</html>

<head>

Призначення: метадані сторінки

Базові атрибути: —

Де розміщується: всередині html

<head>
  <meta charset="UTF-8">
</head>

<body>

Призначення: видимий контент сторінки

Де розміщується: після head

<body>
  Контент
</body>

<div>

Призначення: блочний контейнер

Атрибути: class, id

Де: body

<div class="box">Текст</div>

<p>

Призначення: абзац тексту

<p>Це абзац</p>

<a>

Призначення: гіперпосилання

Атрибути: href, target

<a href="https://example.com">Посилання</a>

<img>

Призначення: зображення

Атрибути: src, alt

<img src="image.jpg" alt="Опис">

<ul> / <ol>

<ul>
  <li>Елемент</li>
</ul>

<table>

<table>
  <tr><td>A</td></tr>
</table>

<b> / <i>

<b>Жирний</b> <i>Курсив</i>

Розміщення сайту на GitHub Pages

  1. Створи репозиторій
  2. Додай index.html
  3. Settings → Pages → Branch: main
  4. Адреса: username.github.io/repo

Ергономіка та поради

  • Одна колонка для читання
  • Sidebar для навігації
  • Контрастний текст
  • Обмеження ширини (~70–80 символів)
